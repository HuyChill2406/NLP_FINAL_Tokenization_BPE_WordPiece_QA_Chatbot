# NLP_FINAL_Tokenization_BPE_WordPiece_QA_Chatbot
This repo contains two NLP assignments: (1) comparing BPE vs non-BPE tokenization on SMS spam classification using TF-IDF + Naive Bayes, and (2) building a simple Vietnamese travel customer-support Q&amp;A bot using a neural text classifier and BLEU-based evaluation.
